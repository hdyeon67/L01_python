# L01_python
> 파이썬 강의 진행 코드

파이썬 기초 문법부터 실무 데이터 가공을 위한 기초 체력을 기르는 과정입니다.   
강의 일차별 코드와 종합 실습 문제가 포함되어 있습니다.

## 📂 디렉토리 구조

```text
.
├── day01/             # 기초 문법: 데이터 타입, 변수, 입출력, 연산자, 조건문
├── day02/             # 반복문 및 기초 컨테이너: List, Tuple, String, For/While 문
├── day03/             # 심화 자료구조 및 알고리즘: Dict, Set, 멤버십 연산, 기초 알고리즘
├── day04/             # 함수 및 메서드: 사용자 정의 함수, Lambda, 자료형별 주요 메서드
└── python_practice/   # 종합 실습 문제 (Q1 ~ Q12)
```

## 📅 커리큘럼 요약

### [Day 01](day01/)
> 파이썬 시작하기
- 데이터 타입과 변수의 개념 이해
- 표준 입출력(`input`, `print`) 활용
- 산술/비교/논리 연산자 및 제어문(`if-else`) 기초

### [Day 02](day02/)
> 데이터의 묶음과 흐름 제어
- 시퀀스 자료형(`List`, `Tuple`, `String`)의 특징과 슬라이싱
- 반복문(`for`, `while`)을 활용한 데이터 순회
- `range()` 함수를 이용한 범위 생성

### [Day 03](day03/)
> 효율적인 데이터 관리와 로직
- 비시퀀스 자료형(`Dict`, `Set`) 및 멤버십 연산자(`in`) 활용
- 다차원 자료구조 탐색 및 데이터 추출
- 기초 알고리즘(최대/최소값 찾기) 및 루프 제어(`break`, `continue`)

### [Day 04](day04/)
> 코드의 재사용과 메서드 활용
- 함수의 정의, 매개변수(`Parameter`)와 반환값(`Return`)
- 익명 함수(`Lambda`)의 활용
- 각 자료형별 필수 메서드(`List`, `String`, `Dict`, `Set`) 학습
